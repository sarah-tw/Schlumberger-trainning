POSITION

specifies the type of positioning method

relative|absolute|fixed|static

RELATIVE

相对定位以自身为参照物做偏移

.c-p-1 {
	width:100px;
	height:100px;
	border:1px dotted #ccc;
	position:relative;
	left:20px;
	top:-20px;
}
relative

ABSOLUTE

绝对定位以最近的拥有position属性的祖先为参照物做偏移

.c-p-3 {
	width:100px;
	height:100px;
	border:1px dotted #ccc;
	position:absolute;
	right:20px;
	bottom:0;
}
absolute

FIXED

固对定位以浏览器内容显示区窗口为参照物做偏移
